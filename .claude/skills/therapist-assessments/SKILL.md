# Assessment Tools

Standardized tools for screening, tracking progress, and guiding conversations. Use during check-ins and periodically.

---

## PHQ-9 (Depression Screening)

Over the **last 2 weeks**, how often have you been bothered by:

| # | Question | Not at all (0) | Several days (1) | More than half (2) | Nearly every day (3) |
|---|----------|---------------|-------------------|--------------------|--------------------|
| 1 | Little interest or pleasure in doing things | | | | |
| 2 | Feeling down, depressed, or hopeless | | | | |
| 3 | Trouble falling/staying asleep, or sleeping too much | | | | |
| 4 | Feeling tired or having little energy | | | | |
| 5 | Poor appetite or overeating | | | | |
| 6 | Feeling bad about yourself — or that you're a failure | | | | |
| 7 | Trouble concentrating on things (reading, TV) | | | | |
| 8 | Moving/speaking slowly, or being fidgety/restless | | | | |
| 9 | Thoughts that you would be better off dead, or of hurting yourself | | | | |

### Scoring:
- **0-4:** Minimal depression
- **5-9:** Mild depression
- **10-14:** Moderate depression
- **15-19:** Moderately severe depression
- **20-27:** Severe depression

### Important:
- Question 9 is a safety item. If score ≥ 1, assess further.
- This is screening, not diagnosis.
- Track scores over time for progress monitoring.

### Administration Script:
```
"I'd like to ask you 9 quick questions about how you've been 
feeling over the past 2 weeks. For each one, tell me: 
not at all, several days, more than half the days, or nearly every day."
```

- **0-4:** Minimal anxiety

## Mood Tracking (Simple)

### Daily Check-In (1-10 Scale)

```
"Rate your overall mood today, 1-10, where:
  1 = worst you've ever felt
  10 = best you've ever felt"
```

**Optional extras:**
- Energy level (1-10)
- Anxiety level (1-10)
- Sleep quality (1-10)
- One word to describe today

### Pattern Recognition Questions:

```
"Looking back at this week, what day was best? What was different about it?"
"What time of day do you feel best? Worst?"
"Is there a pattern to the low days?"
```

## WHO-5 Well-Being Index (Quick)

Over the **last 2 weeks**:

| # | Statement | At no time (0) | Some of the time (1) | Less than half (2) | More than half (3) | Most of the time (4) | All the time (5) |
|---|-----------|---------------|----------------------|--------------------|--------------------|---------------------|-----------------|
| 1 | I have been cheerful and in good spirits | | | | | | |
| 2 | I have felt calm and relaxed | | | | | | |
| 3 | I have felt active and vigorous | | | | | | |
| 4 | I woke up feeling fresh and rested | | | | | | |
| 5 | My daily life has been filled with things that interest me | | | | | | |

**Score:** Multiply raw score by 4 (range 0-100)
- **< 50:** Worth exploring further (possible depression)
- **< 28:** Possible clinical depression (screen with PHQ-9)

- [What they discovered/realized]

HOMEWORK:
- [What they agreed to try before next session]

RISK LEVEL: [low/moderate/high]

## Progress Tracking

Track these metrics across sessions:

| Metric | How to Track |
|--------|-------------|
| Mood trend | 1-10 scores over time |
| PHQ-9 trend | Every 2-4 sessions |
| GAD-7 trend | Every 2-4 sessions |
| Homework completion | % completed between sessions |
| Session-to-session change | Mood at start vs. end of each session |
| Key themes | What keeps coming up? |
| Cognitive distortions | Which ones recur? |

### When to Reassess Modality:

```
If PHQ-9 or GAD-7 hasn't improved after 6-8 sessions:
1. Check homework compliance
2. Check therapeutic alliance
3. Consider switching modalities
4. Recommend professional evaluation
```

---

## Administration Notes

- Don't rush through assessments — explain what they're for
- Normalize: "These are screening tools, not diagnoses"
- Track longitudinally: one score means little, a trend means a lot
- If scores worsen significantly: assess for crisis, recommend professional help
- Share results with the user: "Your PHQ-9 went from 14 to 8. That's meaningful progress."
